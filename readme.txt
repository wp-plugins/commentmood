This plugin allow your reader to post comment with her mood. You can display some chart to get the general mood of a post.

== Description ==

CommentMood allow your visitors to add their mood with comments.
You can customize the display of their mood and add some charts via the option page.
The plugin can be activable by post.

-------------------------------------

CommentMood permet aux visiteurs de votre blog d'ajouter leur humeur en plus de leur commentaire.
Vous pouvez paramétrer l'affichage de ces humeurs et voir des statistiques.
Le plugin est activable par article.

== Installation ==

1. Upload `CommentMood.php` and 'CommentMood.css to the `wp-content/plugins/CommentMood` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can configure it through the "CommentMood" tab in Options.
